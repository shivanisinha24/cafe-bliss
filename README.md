# Cafe Bliss Website

Cafe Bliss is a multi-page website built with **HTML, CSS, and JavaScript**.  
It includes:
- Home page with highlights and "Why Choose Us"
- About Us page with café story
- Menu page (Desserts, Coffee, Main Course)
- Contact page with details and map
- Login and Register forms with validation
